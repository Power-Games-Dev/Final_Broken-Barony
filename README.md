# Final_Broken Barony
This game is a project for my final in a C++ class. 
It's inspired by games like Diplomacy with a fun twist that I thought would make it feel more tactical. 
The introduction of being able to hire spies but at the risk that they may lie to you. 
It's set in a medieval fantasy setting inspired by games I enjoy like, including 
Dungeons and Dragons and World of Warcraft.
